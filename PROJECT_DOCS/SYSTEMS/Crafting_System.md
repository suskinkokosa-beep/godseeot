# Система Крафта

## Обзор

Система крафта Isleborn Online позволяет создавать оружие, броню, инструменты, корабли, постройки и алхимические предметы.

## Категории рецептов

1. **WEAPONS** - Оружие (мечи, копья, луки)
2. **ARMOR** - Броня (шлемы, нагрудники, поножи)
3. **TOOLS** - Инструменты (удочки, кирки)
4. **SHIPS** - Корабли (плоты, лодки, корабли)
5. **BUILDINGS** - Постройки (костры, мастерские, верфи)
6. **ALCHEMY** - Алхимия (зелья, травы)

## Структура рецепта

```gdscript
{
    "id": "stone_knife",

        {"item_id": "stick", "amount": 1}

    "required_skill_level": 1,
    "level": 1
}
```

## Требования

### Ингредиенты
- Все указанные ингредиенты должны быть в инвентаре
- Количество каждого ингредиента должно быть достаточным

### Постройки
- Некоторые рецепты требуют определённую постройку
- Например, корабль можно создать только на верфи

### Уровень навыка
- Рецепты могут требовать минимальный уровень навыка крафта
- С повышением уровня открываются новые рецепты

## Примеры рецептов

### Оружие
- **Каменный нож**: 2 камня + 1 палка (2 сек)
- **Копьё**: 2 палки + 3 камня + 1 верёвка (5 сек)
- **Деревянный меч**: 3 пальмовой древесины + 2 верёвки + 1 камень (8 сек)
- **Железный меч**: 5 металлических слитков + 2 пальмовой древесины + 1 верёвка (30 сек)

### Корабли
- **Плот**: 10 пальмовой древесины + 2 верёвки (30 сек, верфь)
- **Каноэ**: 15 пальмовой древесины + 5 верёвок + 2 ткани (45 сек, верфь)
- **Рыбацкая лодка**: 25 пальмовой древесины + 8 верёвок + 5 ткани + 2 металлических слитка (60 сек, верфь)

### Постройки
- **Костёр**: 5 камней + 3 палки (5 сек)
- **Мастерская**: 20 пальмовой древесины + 15 камней + 5 верёвок (120 сек)
- **Верфь**: 30 пальмовой древесины + 20 камней + 10 верёвок + 5 ткани (180 сек)

## Процесс крафта

1. Выбор рецепта
2. Проверка наличия всех ингредиентов
3. Проверка требования к постройке
4. Проверка уровня навыка
5. Ожидание времени крафта
6. Получение результата
