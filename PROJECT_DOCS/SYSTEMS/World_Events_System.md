# Система Мировых Событий

## Обзор

Система мировых событий Isleborn Online создаёт динамические глобальные события, затрагивающие весь сервер.

## Типы событий

1. **KRAKEN_INVASION** - Нашествие Кракенов
   - Элитные монстры появляются в океане
   - Редкие ресурсы становятся доступнее
   - Высокая опасность

2. **LEVIATHAN_STORM** - Бури Левиафана
   - Гигантская тень размером с биом
   - Элитные монстры и редкие ресурсы
   - Штормовая погода

3. **SIREN_ATTACK** - Сирены-душегубки
   - Атакуют корабли ночью в тумане
   - Лут от сирен
   - Репутация с Морской Стражей

4. **ICE_STORM** - Ледяной шторм
   - Морозная буря
   - Ледяные монстры
   - Замедление движения

5. **AI_GUILD_UPRISING** - Восстание AI-гильдий
   - AI-гильдии атакуют игроков
   - Рейды на острова
   - Гильдийные войны

6. **VOID_PORTAL** - Открытие порталов бездны
   - Blackwater-монстры на поверхности
   - Зоны искажения
   - Экстремальная опасность

7. **SEASONAL_EVENT** - Сезонное событие
   - Особые события по сезонам
   - Эксклюзивные награды
   - Косметика

## Механика

- События объявляются заранее (60 секунд до старта)
- События активны определённое время
- Все игроки могут присоединиться
- Прогресс события отслеживается совместно

## Эффекты

События влияют на:
- Погоду
- Спавн монстров
- Доступность ресурсов
- Видимость
- Скорость движения

## Награды

- Множитель опыта
- Редкие предметы
- Уникальный лут
- Репутация с фракциями
- Косметика (сезонные)
