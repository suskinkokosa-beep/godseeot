# Система Экспедиций

## Обзор

Система экспедиций Isleborn Online позволяет игрокам отправлять NPC, корабли и флоты в долгосрочные миссии, которые выполняются в реальном времени.

## Типы экспедиций

1. **GATHER_RESOURCES** - Собрать ресурсы
   - Требования: грузоподъёмность, экипаж
   - Награды: ресурсы, опыт, валюта
   - Риски: атаки монстров, штормы

2. **EXPLORE_BIOME** - Исследовать биом
   - Требования: скорость корабля
   - Награды: знание биома, опыт, валюта
   - Риски: встреча с монстрами

3. **HUNT_MONSTERS** - Охота на монстров
   - Требования: вооружение, боевой экипаж
   - Награды: части монстров, опыт
   - Риски: ранения, повреждение корабля

4. **FIND_TREASURE** - Поиск сокровищ
   - Требования: снаряжение для дайвинга
   - Награды: редкие предметы, золото
   - Риски: опасные монстры, потеря экипажа

5. **TRADE_ROUTE** - Торговый маршрут
   - Требования: грузовое пространство, скорость
   - Награды: валюта, репутация
   - Риски: пиратские атаки, потеря груза

6. **BLACKWATER_RIFT** - Исследование Blackwater разлома
   - Требования: высокий уровень, магическое оборудование
   - Награды: редкие ресурсы, большой опыт
   - Риски: экстремальная опасность, потеря всего

## Механика

- Экспедиции выполняются в **реальном времени**
- Игрок может быть оффлайн во время экспедиции
- Прогресс отслеживается автоматически
- Риски могут привести к провалу экспедиции

## Награды

Зависит от типа экспедиции:
- Ресурсы
- Опыт персонажа
- Валюта (ракушки, золото)
- Редкие предметы
- Репутация с фракциями

## Риски

- Атаки монстров
- Штормы
- Потеря экипажа
- Повреждение/уничтожение корабля
- Провал экспедиции
