# Краткое описание системы навыков

## Основные концепции

### Получение навыков
- Навыки изучаются из **книг навыков**
- Книги можно получить из:
  - Дропа монстров (зависит от тира)
  - Наград за квесты
  - Покупки у NPC
  - Находок в руинах
  - Дропа боссов

### Использование навыков
- **5 слотов** для активных навыков
- **3 слота** для пассивных навыков
- Активные навыки требуют маны/стамины и имеют кулдаун
- Пассивные навыки работают постоянно

### Ограничения
- Активные навыки привязаны к классу
- Пассивные навыки универсальны
- Игрок не может использовать навыки, запрещённые его классом

### Улучшение
- 1 очко навыка = 1 уровень
- Каждые 5 уровней персонажа = +1 очко навыка
- Эффекты масштабируются с уровнем

## Категории активных навыков

1. **Ближний бой** (15 навыков)
2. **Дальний бой** (15 навыков)
3. **Магия** (20 навыков)
4. **Алхимия** (15 навыков)
5. **Мореплавание** (15 навыков)
6. **Собирательство** (10 навыков)
7. **Защита** (5 навыков)
8. **Движение** (5 навыков)

**Всего: ~100 активных навыков**

## Пассивные навыки

- Универсальные для всех классов
- ~50 пассивных навыков
- Улучшают характеристики, дают бонусы, добавляют новые способности
