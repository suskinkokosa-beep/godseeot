# Обзор базы данных предметов

## Категории предметов

### Ресурсы
- Пальмовая древесина
- Камень
- Палка
- Верёвка
- Ткань
- Металлический слиток

### Оружие ближнего боя
- Каменный нож (Common)
- Копьё (Common)
- Деревянный меч (Common)
- Железный меч (Uncommon)

### Оружие дальнего боя
- Деревянный лук (Common)
- Арбалет (Uncommon)

### Щиты
- Деревянный щит (Common)

### Броня
- Тканевый комплект (Common)

  - Рубаха
  - Штаны
  - Сапоги
  - Перчатки
- Кожаный комплект (Common)

  - Доспех

### Аксессуары
- Медное кольцо (Common)
- Кожаный амулет (Common)

### Инструменты
- Удочка
- Кирка

### Расходники
- Целебная трава (восстанавливает HP)

## Статы предметов

Все экипируемые предметы имеют:
- Базовые статы (damage, defense, etc.)
- Поддержку заточки (0-12)
- Бонусы от заточки (+5% за уровень)

## Редкость предметов

- Common (Обычный)
- Uncommon (Необычный)
- Rare (Редкий)
- Epic (Эпический)
- Legendary (Легендарный)
