# Система Классов и Навыков Персонажа

## Обзор

Isleborn Online использует систему прогрессии без классов до 10 уровня, после чего игрок выбирает свой архетип. Система навыков основана на изучении книг навыков.

## Прогрессия Персонажа

### Уровни

- **1-9 уровень**: Без класса, универсальные навыки
- **10 уровень**: Выбор архетипа (класса)
- **Каждые 5 уровней**: 5 очков статов + 1 очко навыка

### Статы

8 основных характеристик:

1. **Сила (Strength)**: Урон в ближнем бою, переносимый вес
2. **Живучесть (Vitality)**: Максимальное здоровье
3. **Ловкость (Agility)**: Скорость движения, уклонение
4. **Выносливость (Stamina)**: Максимальная стамина
5. **Фокус (Focus)**: Скорость сбора ресурсов
6. **Интеллект (Intelligence)**: Мана, эффективность крафта
7. **Восприятие (Perception)**: Заметность врагов, точность
8. **Удача (Luck)**: Шанс крита, качество дропа

## Архетипы (Классы)

### 1. Собиратель (Gatherer)
- **Описание**: Мастер добычи ресурсов и исследования океана
- **Бонусы статов**: Focus +3, Stamina +2
- **Разрешённые навыки**: Собирательство, исследование, торговля
- **Запрещённые навыки**: Тяжёлый бой, наступательная магия

### 2. Боец (Fighter)
- **Описание**: Великий воин ближнего боя
- **Бонусы статов**: Strength +5, Vitality +3
- **Разрешённые навыки**: Ближний бой, защита, берсерк
- **Запрещённые навыки**: Дальний бой, магия

### 3. Стрелок (Ranger)
- **Описание**: Мастер дальнего боя и точности
- **Бонусы статов**: Agility +5, Perception +3
- **Разрешённые навыки**: Дальний бой, охота, ловушки
- **Запрещённые навыки**: Тяжёлая броня, магия

### 4. Моряк (Sailor)
- **Описание**: Искусный мореплаватель и капитан
- **Бонусы статов**: Seamanship +5, Leadership +3
- **Разрешённые навыки**: Мореплавание, корабельный бой, навигация
- **Запрещённые навыки**: Нет

### 5. Алхимик (Alchemist)
- **Описание**: Мастер зелий и взрывчатых веществ
- **Бонусы статов**: Intelligence +5, Crafting +3
- **Разрешённые навыки**: Алхимия, бомбы, зачарование
- **Запрещённые навыки**: Тяжёлый бой

### 6. Мистик (Mystic)
- **Описание**: Владелец тайной магии океана
- **Бонусы статов**: Magic +5, Wisdom +3
- **Разрешённые навыки**: Магия, стихии, призыв
- **Запрещённые навыки**: Физический бой

### 7. Строитель (Builder)
- **Описание**: Мастер строительства и архитектуры
- **Бонусы статов**: Construction +5, Engineering +3
- **Разрешённые навыки**: Строительство, автоматизация, защитные структуры
- **Запрещённые навыки**: Наступательный бой

## Система Навыков

### Типы навыков

1. **Активные навыки**: 5 слотов для экипировки
2. **Пассивные навыки**: 3 слота для экипировки

### Редкость навыков

- **Обычный (Common)**: Базовые навыки, легко получить
- **Редкий (Rare)**: Сильные навыки, средняя редкость
- **Легендарный (Legendary)**: Мощные навыки, редкие
- **Мифический (Mythic)**: Уникальные навыки, крайне редкие

### Получение навыков

Навыки изучаются из **книг навыков**, которые можно получить:

1. **Дроп с монстров**: Зависит от тира монстра
2. **Награды за квесты**: Определённые квесты дают книги
3. **Покупка у NPC**: Некоторые торговцы продают книги
4. **Находки в руинах**: Редкие книги в древних руинах
5. **Дроп с боссов**: Мифические книги только с боссов

### Категории активных навыков

- **Ближний бой (Melee)**: 15 навыков
- **Дальний бой (Ranged)**: 15 навыков
- **Магия (Magic)**: 20 навыков
- **Алхимия (Alchemy)**: 15 навыков
- **Мореплавание (Sailing)**: 15 навыков
- **Собирательство (Gathering)**: 10 навыков
- **Защита (Defense)**: 5 навыков
- **Движение (Movement)**: 5 навыков

**Всего активных навыков**: ~100

### Пассивные навыки

- **Универсальные**: Могут использовать все классы
- **Всего пассивных навыков**: ~50

### Улучшение навыков

- Используется **1 очко навыка** для повышения уровня
- Каждый навык имеет максимальный уровень (обычно 1-5)
- Эффекты масштабируются с уровнем навыка

### Забывание навыков

- Игрок может забыть изученный навык
- Навык удаляется из списка изученных
- Очки навыков не возвращаются

## Ограничения по классам

- **Активные навыки**: Ограничены классом персонажа
- **Пассивные навыки**: Универсальны для всех классов
- Игрок не может использовать активные навыки, запрещённые его классом

## Примеры навыков

### Активные (Ближний бой)

- **Рез (Common)**: Базовая атака мечом
- **Вихрь (Rare)**: Вращающаяся атака по всем врагам вокруг
- **Кровожадность (Legendary)**: Каждая атака восстанавливает здоровье
- **Рассечение бездны (Mythic)**: Мощная атака с тёмной магией

### Пассивные

- **Усиление живучести (Common)**: +50 к максимальному здоровью
- **Регенерация (Rare)**: Медленно восстанавливает здоровье
- **Чувство океана (Legendary)**: Мини-карта показывает монстров
- **Бессмертная душа (Mythic)**: 20% шанс воскреснуть после смерти

## Формулы

### Опыт для уровня
```
XP = 50 * Level^1.9
```

### Награды каждые 5 уровней
```
Статы: +5 очков
Навыки: +1 очко
```

### Выбор класса
```
Уровень >= 10
Класс не выбран
```
